# 音色脚本生成指南 (voices_script.txt)

从 `bible/characters.md` 读取人物档案，生成 `bible/voices_script.txt`，用于 TTS 基线音色生成。

## 格式

与压缩稿格式一致：`角色名(音色描述)：台词`

每行一个角色。`characters.md` 中「核心人物」+「弧级人物」都需要生成，路人和龙套跳过。旁白也需添加。

## 音色描述规则

必须从人物档案提炼出**声音特质**（物理属性），不是某句台词的念法：

- **性别 + 年龄感**：少年/中年/老年男性/女性
- **声线特征**：清澈/沙哑/浑厚/尖锐/偏薄/中气足
- **语速倾向**：平稳/偏快/偏慢/每个字咬得清
- **气息特点**：气息重/气声多/声音发紧/音量大小

**与章节对话中的语气描述是两回事**：
- 音色描述 → 角色**固有的声音设定**（`少年男性，声音清澈`）
- 语气描述 → 某句台词**当时的念法**（`声音沉下去`、`尾音往上甩`），见 emotion_tags.md

## 台词

写一段 **40~60 字**的中性叙述句，用于展示角色的声音特质。

推荐格式：简单自我介绍 + 一句不涉及具体剧情的中性陈述。不要太短（参考音频太短克隆效果差）。

## 示例

从 `characters.md` 提取的人物档案：

```
### 庆尘
- 身份：17岁高中生
- 性格：早熟冷静，言语犀利
```

生成：

```
庆尘(17岁少年男性，声音清澈干净，语速平稳，冷静克制)：我叫庆尘，今年十七岁，是洛城一名普通的高中生。我不太爱说话，但该说的，我一句都不会少。
```

再如：

```
### 张大爷
- 身份：50岁老头
- 性格：骂骂咧咧但不真恼
```

生成：

```
张大爷(50岁中老年男性，声音略带沙哑，中气尚足，带市井味)：我叫老张，街坊都喊我张大爷。五十出头，身上没啥本事，就爱在胡同口跟人杀两盘棋。
```

## 旁白

叙述者也需要音色，推荐：

```
旁白(成年男性旁白声线，沉稳清晰，不疾不徐，不带个人感情)：我在这里，只是把看到的事情，原原本本地讲给你们听。不添油加醋，也不替谁遮掩。
```

## 常见遗漏

- **忘了旁白**：旁白也需要基线音色，不然每次遇到旁白行都会走声音设计降级
- **台词太短**：`"你好，我是庆尘。"` 才 8 个字，生成出的参考音频不足 1 秒，克隆效果差
- **音色描述写成了语气**：`声音沉下去，很平` 是语气不是音色；音色是 `声音清澈，语速平稳`
