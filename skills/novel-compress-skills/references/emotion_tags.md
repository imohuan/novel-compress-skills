# Voice Design 描述指南

> 基于 VoxCPM 2 的 Voice Design 模式。核心原则：**描述声音的物理属性，不描述角色的心理状态。**

## 铁律

描述必须回答"这声音**听起来**什么样"，而不是"这角色**心里在想**什么"。

**❌ 禁止的抽象词汇**（AI 无法转化为声学参数）：

不甘心、无奈、失落、期待、急切、好奇、盘算、试探、不屑、戒备、庆幸、感慨、犹豫、为难、压抑、释然、真诚、诚恳、敷衍、轻蔑、失望、欣慰、愧疚、自责、得意、自信、自卑、心虚、隐忍、克制

**✅ 能用的物理描述**：

| 维度 | 可用描述 | 效果 |
|------|----------|------|
| **音高** | 声音拔高、声音沉下去、尖锐、低沉 | 音高是表意最强的声学参数 |
| **语速** | 一字一顿、连珠炮、拖长尾音、越说越快、说到一半停住 | 语速=紧张度 |
| **音量** | 压低声音、气声、几乎听不见、音量突然拔高 | 音量=控制感 |
| **气息** | 叹气开头、带着气声、气息不稳、上气不接下气 | 气息=情绪泄漏 |
| **颤抖** | 声音发抖、尾音发颤、全程都在颤 | 颤抖=恐惧/激动 |
| **发紧** | 嗓子发紧、声音挤出来的、喉咙像被掐住 | 发紧=压抑 |
| **断裂** | 说到一半断了、哽咽、吞咽声后继续、一句话分成几截 | 断裂=崩溃边缘 |
| **拖音** | 尾音拖长、每个字都拉得特别长、带着鼻音拖腔 | 拖音=不情愿/撒娇 |
| **短促** | 每个字都短、几乎听不出尾音、像在往下砍 | 短促=愤怒/命令 |

---

## 输出格式

```
角色名(声音听起来什么样)：对话内容
旁白：叙述内容
```

## 实战：旧写法 → 新写法

```
❌ 庆尘(平静)                        → ✅ 庆尘(声音沉下去，几乎听不出情绪)
❌ 张大爷(不甘心)                     → ✅ 张大爷(声音发抖，尾音拖长，像叹气)
❌ 庆尘(带着不易察觉的嘲讽)            → ✅ 庆尘(尾音微微上扬，一字一顿)
❌ 魁梧囚犯(挑衅)                     → ✅ 魁梧囚犯(低沉，语速放慢，尾音上扬带着笑)
❌ 黄济先(崩溃)                       → ✅ 黄济先(声嘶力竭，嗓音劈叉，语无伦次)
❌ 机械眼老头(戏谑)                   → ✅ 机械眼老头(笑着说的，语速不急但每个字都咬得很清)
❌ 李叔同(温润如玉)                   → ✅ 李叔同(男中音，不急不缓，每个字都很稳)
❌ 庆尘(冷到骨子里)                   → ✅ 庆尘(压得极低，几乎是从牙缝里挤出来的，没有任何尾音)
❌ 庆尘(一字一顿，强作镇定)            → ✅ 庆尘(语速异常平稳，每个字间隔一致，像在刻意控制)
❌ 林小笑(意味深长)                    → ✅ 林小笑(尾音拖长但不降调，像在等人接话)
❌ 南庚辰(神秘兮兮)                    → ✅ 南庚辰(气声，音量压得很低，偶尔往外偷瞄的气息)
```

## 角色声线模板（物理属性版）

| 角色类型 | 声线描述 |
|----------|----------|
| 庆尘（少年主角） | 年轻人声线，偏薄偏清，但习惯于把声音压平 |
| 庆尘冷静时 | 声音沉下去，语速稳，尾音干净 |
| 庆尘狠起来 | 压到极低，从牙缝里挤出来的，没有尾音 |
| 张大爷（老年配角） | 老年人声线，气息重，每个字之间有换气的停顿 |
| 李叔同（中年权威） | 中年人声线，男中音，语速偏慢，每个字都稳 |
| 南庚辰（话唠同桌） | 年轻人声线偏尖，语速比正常人快三分之一，句子挤在一起 |
| 林小笑（机灵随从） | 年轻人声线偏高，语速快但每个字都很清，经常尾音上扬 |
| 叶晚（沉稳保镖） | 年轻人声线偏低偏厚，说话像每句都在脑子里先过一遍 |
| 路广义（话唠舔狗） | 年轻人声线，句子连句子几乎没有间隔，声音里总有笑意 |
| 江雪（年轻女性） | 女性声线偏柔，气息偏多，像不太敢用力说话 |
| 黄济先（崩溃） | 声嘶力竭，嗓音已经劈叉了，句子碎成一片一片 |
| 广播/AI | 女性声线，语速完全均匀，句间间隔精确一致，没有任何感情波动 |
| 小姑娘 | 音调偏高，气息偏弱，句尾习惯性上扬 |

## 层层递进的情绪

同一章内同一角色心情逐步升级时，描述也要跟着升级：

```
庆尘(声音沉下去，很平)：将军。
↓ 张大爷赖账
庆尘(声音沉下去，但尾音开始上扬)：不用挣扎了。
↓ 张大爷又赖账
庆尘(语速明显变快，每个字都短，像在砍)：一星期学会就能找回面子。
↓ 说到"他下的也不怎么样"
庆尘(忽然放慢，尾音上扬，带着一点刚藏进去的笑)：毕竟他下的也不怎么样。
```

## VoxCPM 底层

```python
wav = model.generate(
    text="(年轻人声线，声音沉下去，几乎听不出情绪)将军。",
    cfg_value=2.0,
    inference_timesteps=10,
)
```

- `cfg_value` 默认 2.0，追求更自然降到 1.5-1.6
- 长文本分段合成后拼接
- 中英混合描述均可，如 `(slightly faster, voice cracking at the end)`
