# 写作手法知识库模板

> 此文件由每弧结束时自动生成/追加。格式设计用于 AI 精准复刻。

## 使用说明

每条技法包含：
- **名称**：简短标签
- **定义**：一句话说明这个手法
- **示例**：原文具体案例 + 章节来源
- **复刻 Prompt**：可直接粘贴给 AI 的提示词片段
- **适用场景**：什么时候用这个手法
- **配合手法**：和哪些手法一起用效果最好

## 技法1：环境作为情绪外化

- **定义**：天气/场景不是装饰，是人物心理的延伸
- **类别**：场景与氛围
- **示例(ch001)**：
  > "雨棚之外的全世界灰暗，地面被雨水沁成了黑色，只有雨棚下的地面还留着一片干燥地带，就像是整个世界都只剩下这一块净土。"
  - 功能：孤独感、与世界隔绝的疏离
- **复刻 Prompt**：
  ```
  场景描写不是为了美丽，是为了暴露人物内心状态。
  规则：孤独用冷色调+狭小空间+雨天/夜晚；愤怒用高温+暴晒+拥挤；
  幸福用暖色调+开阔+日光。一句场景可能就够了，但必须是心理外化。
  ```
- **适用场景**：开篇、角色低谷时刻、转折前夕
- **配合手法**：延迟揭示、对比反差

## 技法2：对话替代旁白交代信息

- **定义**：角色信息（性格/背景/需求）完全通过对白透露，不用叙述段落
- **类别**：对话写作
- **示例(ch001)**：
  > 老头："每天都要输你20块钱！" → 交代了交易规则
  > 庆尘："但我还教你下棋找他们赢回面子" → 交代了庆尘的谋生方式
- **复刻 Prompt**：
  ```
  禁止用"张三是个冷漠的人"这类旁白。用对话行动展示：
  张三(对服务员)：不用。 → 一句话比三百字介绍有效。
  每条对话承担至少两个功能：推进情节+展示性格+传递信息。
  ```
- **适用场景**：开篇人物出场、关系确立
- **配合手法**：信息不对称、延迟揭示

## 技法3：延迟揭示

- **定义**：关键信息推迟到段落/章节末尾才揭晓，制造"先猜后懂"的阅读体验
- **类别**：悬念与伏笔
- **示例(ch001)**：
  > 前面一直在写"庆尘在等人"，读者猜测等恋人/朋友
  > 结尾揭晓：母亲带着新家庭经过，庆尘转身离开。"他要等的人来了，但他又不想等了。"
- **复刻 Prompt**：
  ```
  不要在开篇揭示动机。用模糊代称（"那个人""那件事"）建立悬念，
  在段落末尾用反差揭示真相。反差越大，冲击越强。
  ```
- **适用场景**：开篇钩子、转折点
- **配合手法**：视角转换、情感操纵
