# 剧情圣经（Story Bible）维护规范

bible/ 是跨章真相源，抗"人设崩塌/静默漂移"的支点。

## 文件
- `characters.md`：人物。每条格式 `- 名字：身份/关键特征（来源：ch0XX）`。不可变身份事实单独标注。
- `settings.md`：世界规则、地点、独门设定、意象。
- `timeline.md`：时间线，按章节顺序记录跳跃。
- `throughline.md`：主线 throughline + 关键钩子清单。

## 规则
- 正文与 bible 冲突时，以 bible 为准，重生成该章。
- 每章压缩后增量回写 delta，不整体重写。
- 关键事实（眼睛颜色、疤痕位置等）存为不可变条目，供 verify 检索。

## 示例（characters.md）
```
- 林惊羽：男主，左眉骨有旧疤（来源：ch003）
- 苏沐雪：女主，擅剑，口头禅"剑无虚发"（来源：ch007）
```
